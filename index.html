Proyecto conceptual de trading automatizado presentado como landing page

Esta página fue creada para practicar maquetación con HTML y CSS, enfocándome en estructura, layout y presentación de una idea técnica.

¿Qué es KNT Capital?

KNT Capital es un proyecto conceptual que representa una empresa ficticia enfocada en trading automatizado. El objetivo del proyecto no es el sistema financiero en sí, sino la práctica de cómo presentar una idea técnica de forma clara y estructurada mediante una página web.

Análisis

El sistema analiza datos del mercado utilizando reglas predefinidas.

Decisión

Con base en esas reglas, se determina si se ejecuta una acción.

Ejecución

Las acciones se ejecutan automáticamente según la estrategia definida.

¿Qué hice yo en este proyecto?

En este proyecto trabajé principalmente en esta pagina utilizando el proyecto del bot de trading como herramienta de presentacion, enfocandome en:

  • Estructura HTML semántica
  • Layout con Flexbox
  • Organización visual de secciones
  • Diseño responsive básico
  • Presentación clara de una idea técnica

Durante este proyecto aprendí a:

  • estructurar una landing desde cero
  • replicar un diseño existente
  • organizar contenido de forma clara
  • usar Flexbox para layouts reales
